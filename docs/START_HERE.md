## 📋 Los 5 Pasos del Refactor

### PASO 1: Setup Core (30 minutos)
Configura Tailwind, CSS variables y el layout principal.

### PASO 2: Components Base (2 horas)
Actualiza: button, card, input, select

### PASO 3: Components Secundarios (1 hora)
Actualiza: dialog, table, tabs

### PASO 4: Components Comunes (1 hora)
Crea: EmptyState, Alert, Badge, LoadingSpinner

### PASO 5: Refactor Páginas (2-3 horas)
Refactoriza todas tus páginas (Dashboard, Patients, etc)

**Total: 6-7 horas**
