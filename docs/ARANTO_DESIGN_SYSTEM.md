## Resumen

- **Modo oscuro:** #0f1a1e con gradientes esmeralda
- **Modo claro:** blanco con acentos esmeralda
- **Color principal:** #34A853 (verde Aranto)
- **Secundarios:** teal, naranja, rojo
- **Tipografía:** Sans-serif profesional, compacta
- **Cards:** Gradientes, bordes redondeados xl/2xl
- **Espaciado:** Grid de 8px
- **Marca:** Salud, confianza, profesionalismo
